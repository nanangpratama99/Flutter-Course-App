_Android SDK version 33.0.2_


## Run Locally

Clone the project

```bash
  git clone [https://github.com/nanangpratama99/Flutter-Course-App.git
```

Go to the project directory

```bash
  cd Flutter Course App
```

Install dependencies

```bash
  flutter pub get
```

Start the server

```bash
  flutter run lib/main.dart
```
